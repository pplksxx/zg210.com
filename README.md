[中股证券](https://zg006.cc/)是由香港中股证券有限公司研发运营的股票配资平台软件，注册号：77632967，提供证券投资与股权投资管理咨询服务。该平台支持股指、A股现货交易及3-10倍杠杆配置，实行资金三方监管，包含免息配资和按天配资两种模式：前者提供30天固定期限免管理费服务，用户可获得盈利的80%；后者按实际天数计算管理费，不含印花税及佣金。
平台要求用户通过券商官网下载独立证券账户进行交易，并提供7×24小时一对一专属服务。运营团队结合互联网开发与金融风险管理经验，构建涵盖业务模式、风险控制及技术保障的体系，致力于打造安全灵活的证券交易环境。中股证券以“专业、诚信、创新、共赢”为服务宗旨，专注提升服务质量与用户价值。
平台简介
平台提供股指、A股现货交易及3-10倍杠杆服务，支持零门槛开户并实行资金三方监管。该平台提供免息配资和按天配资两种模式：免息配资期限固定为30天，杠杆倍数3-10倍且不收取管理费用；按天配资按实际天数计算费用，管理费不含印花税及佣金。运营团队由互联网公司与证券金融机构专家组成，专注金融风险管理和技术保障。

免息配资
配资期限，免息配资固定配资期限为30天，配资资金：免息配资为3-10倍固定杠杆，所有费用全部免费，不收取管理费盈利80%归用户。
按天配资
配资期限，按天计算，不包含各类法定节假日，按天支付管理费（不包含交易印花税、过户费和佣金），无其他任何费用，如配资2天，一次性收取2天管理费。
